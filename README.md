# BitMeet
> Decentralized, encrypted and serverless Microsoft Teams alternative.

BitMeet is a state-of-the-art P2P video conferencing and chat platform with zero knowledge architecture.

## Features

- **True Privacy**: Peer-to-Peer (P2P) architecture. Your data never touches our servers.
- **Zero knowledge**: Like Proton, we don't store nothing more than your user public key and company.
- **Just like Microsoft Teams or Skype**: Send files via our innovative P2P file sharing system (Bitshare), send emojis, links, schedule calls, etc.
- **Serverless Discovery**: Intelligent participant discovery without any server.
- **Zero Footprint on chat and calls**: Our server only stores your user public key to make de encrypted chat and calls possible. No chat history or calls stored on any server.
- **E2EE Ready**: Encrypted media streams via WebRTC native protocols.
